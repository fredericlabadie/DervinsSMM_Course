# Legacy / archival files

`smm_guide.qmd` is a legacy single-file version of the course retained in the repository for archival reference only.

It is not rendered by `_quarto.yml` and should not be treated as the maintained source of truth. The maintained live site is built from:

- `index.qmd`
- `bridge.qmd`
- `theory.qmd` plus `theory/*.qmd`
- `method.qmd`
- `fieldwork.qmd`

If a maintained print/PDF version is needed later, create a new `print.qmd` that includes the live modular sections rather than reviving `smm_guide.qmd`.
