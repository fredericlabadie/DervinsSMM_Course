# Theory page modules

`theory.qmd` is intentionally small and includes these section modules in display order. Edit the relevant module instead of editing the full rendered page.

## Display order

- `01-hero.qmd` — page header, central metaphor, abstract front-matter card; visible as Plate I
- `06-sgbo.qmd` — course S-G-B-O teaching shorthand and diagram; visible as Plate II
- `02-assumptions.qmd` — seven-point course synthesis of Dervin's methodological stance; visible as Plate III
- `03-sense-unmaking.qmd` — later sense-unmaking framing and practitioner application; visible as Plate IV
- `04-transmission-model.qmd` — SMM vs. transmission-model comparison; visible as Plate V
- `05-comparative-analysis.qmd` — neighbouring frameworks table; visible as Plate VI
- `07-bibliography.qmd` — verified bibliography / recommended reading; visible as Plate VII
- `08-in-memoriam.qmd` — Dervin memorial note; visible as Plate VIII

The file numbers are not perfectly sequential because `06-sgbo.qmd` was moved earlier in the rendered page to preserve the plate sequence. Do not reorder the wrapper alphabetically.

## Editing rule

Keep the practitioner voice, but label claim status clearly:

- **Academic note** — Dervin's own formulations, chronology, or source caveats
- **Practitioner note** — how the course applies the idea in UX, analytics, product, or fieldwork
- **Course synthesis** — teaching structure drawn across several Dervin sources, not a single direct enumeration
