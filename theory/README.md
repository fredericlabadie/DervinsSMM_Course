# Theory page modules

`theory.qmd` is intentionally small and includes these section modules in order. Edit the relevant module instead of editing the full rendered page.

## Module map

- `01-hero.qmd` — page header, central metaphor, abstract front-matter card
- `02-assumptions.qmd` — seven-point course synthesis of Dervin's methodological stance
- `03-sense-unmaking.qmd` — later sense-unmaking framing and practitioner application
- `04-transmission-model.qmd` — SMM vs. transmission-model comparison
- `05-comparative-analysis.qmd` — neighbouring frameworks table
- `06-sgbo.qmd` — course S-G-B-O teaching shorthand and diagram
- `07-bibliography.qmd` — verified bibliography / recommended reading
- `08-in-memoriam.qmd` — Dervin memorial note

## Editing rule

Keep the practitioner voice, but label claim status clearly:

- **Academic note** — Dervin's own formulations, chronology, or source caveats
- **Practitioner note** — how the course applies the idea in UX, analytics, product, or fieldwork
- **Course synthesis** — teaching structure drawn across several Dervin sources, not a single direct enumeration
